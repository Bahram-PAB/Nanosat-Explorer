# 🛰️ کاوشگر ماهواره‌های کوچک (Nanosat Explorer)

یک اپلیکیشن اندرویدی مدرن، سریع و کاملاً آفلاین برای جستجو، فیلتر پیشرفته و بررسی مشخصات فنی دقیق ماهواره‌های کوچک (Nanosatellites) در سراسر جهان. این برنامه با آخرین استانداردهای توسعه اندروید طراحی شده و از یک پایگاه داده محلی بسیار غنی بهره می‌برد.

---

## 🚀 ویژگی‌های کلیدی (نسخه ۴.۱)

*   **نمایش تک‌ستونه و مدرن (Single Column Layout):** بهبود خیره‌کننده رابط کاربری با تغییر ساختار دو‌ستونه به یک ستون یکپارچه، عریض و بسیار خوانا جهت مرور راحت‌تر لیست ماهواره‌ها.
*   **سیستم نشان‌گذاری و علاقه‌مندی‌ها (Star/Favorite):** امکان نشان‌دار کردن ماهواره‌ها به صورت مستقیم از روی کارت هر ماهواره در لیست اصلی و بررسی سریع آن‌ها از طریق فیلتر برگزیده‌ها.
*   **فیلتر هوشمند با دکمه پاک‌سازی جدید 🧹:** جایگزینی دکمه چرخش سنتی با آیکون «جارو» (Cleaning Services) جهت پاک‌سازیِ سریع فیلترهای اعمال‌شده با ظاهری مدرن‌تر.
*   **محیط کاربری تم تاریک کیهانی (Cosmic Dark Theme):** استفاده از روانشناسی رنگ‌های شب زنده و المان‌های فضایی منطبق بر اصول **Material Design 3** جهت کاهش خستگی چشم.
*   **موتور پردازشگر پایگاه‌داده بسیار پایدار:** مجهز به ضدالگوریتم برطرف‌کننده باگ‌های ساختاری فایل‌های JSON حجیم، برای تضمین پایداری مطلق برنامه در اولین اجرا و ثبت مستقیم آمار در پایگاه‌داده محلی **Room**.
*   **فیلترهای پیشرفته:** فیلترینگ چندگانه بر اساس کشور سازنده، وضعیت فعلی ماهواره (فعال، غیرفعال، سقوط‌کرده)، و دسته‌بندی ابعادی (کلاس‌های U).
*   **امپراتوری ناوبری روان:** استفاده از دکمه بازگشت مستقیم استاندارد اندروید برای مهاجرت روان بین صفحات جزییات و صفحه اصلی کاوشگر.

---

## 🛠️ ساختار فنی و پشته فناوری (Tech Stack)

این پروژه از به‌روزترین معماری‌ها و ابزارهای توسعه پیشنهادی گوگل استفاده می‌کند:

*   **زبان برنامه‌نویسی:** ۱۰۰٪ کاتلین (Kotlin) با مدیریت همروندی Coroutines و Flow.
*   **رابط کاربری (UI):** Jetpack Compose انقلابی، کاملاً اعلامی (Declarative) و منطبق بر تم پویا Material 3.
*   **پایگاه‌داده محلی:** Room Database به همراه کتابخانه پردازش نمادین کاتلین (KSP) برای ذخیره‌سازی همگام سوابق ماهواره‌ها.
*   **معماری:** معماری استاندارد MVVM (Model-View-ViewModel) با جداسازی کامل وظایف لایه‌ها روندهای پاک (Clean Architecture).
*   **تبدیل داده:** بهینه‌شده با ابزار پردازش بومی JSON برای رفع خطاهای کاراکتری بدون اتلاف منابع پردازنده دستگاه.

## 📦 جزئیات نسخه ۴.۱ (Release Sandbox)

*   **کد نسخه (Version Code):** 4
*   **نام نسخه (Version Name):** 4.1
*   **هدف توسعه:** ارتقای سطح دسترسی (Accessibility)، بازطراحی بصری صفحات لیست بر اساس بازخوردهای ارگونومی، و پیاده‌سازی مکانیزم نشان‌گذاری منعطف ماهواره‌ها.

## 📸 نمایی از محیط برنامه

*   **صفحه اصلی:** لیست ماهواره‌ها به صورت عمودی با امکان نشان‌دار کردن سریع، نوار بالای کیهانی به همراه دکمه جاروب فیلترها و کادر جستجوی زنده.
*   **صفحه جزئیات:** کارت مشخصات فنی دقیق اعم از جرم، ابعاد، تاریخ پرتاب، مدار و کشور سازنده با دکمه بازگشت ارگونومیک.
